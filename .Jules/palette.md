## 2026-04-19 - Accessible Material Symbols Icon Buttons
**Learning:** In this application, many interactive buttons consist solely of a Material Symbols icon (using the `material-symbols-outlined` class) without visible text. Without proper ARIA attributes, screen readers read out the literal text content (the ligature name, e.g., 'search', 'picture_as_pdf', 'arrow_upward'), which can be confusing or lack context, and they don't announce the element's purpose clearly.
**Action:** Always follow the pattern of adding a descriptive `aria-label` to the parent `<button>` element and `aria-hidden="true"` to the inner `<span>` containing the icon ligature. This ensures screen readers announce the intended action (e.g., 'Search', 'Export PDF') while ignoring the visual icon text.
