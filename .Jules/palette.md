## 2024-05-14 - Empty States Should Drive Engagement
**Learning:** Empty states in dynamic sections (like recent sessions or history) without a clear Call to Action (CTA) lead to user drop-off. Text-only messages like "No recent sessions" represent a missed opportunity to guide users toward the core value of the application.
**Action:** When creating or updating dynamic content views, always provide an empty state card that includes a description of what will appear there and a primary CTA button that navigates the user to the place where they can generate that content.
