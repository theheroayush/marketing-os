## 2023-10-24 - Actionable Empty States
**Learning:** Text-only empty states create dead-end user flows, halting engagement. Replacing them with visually distinct components that include explicit Call-To-Action (CTA) buttons provides a clear path forward and encourages continued interaction.
**Action:** Always implement empty states for dynamic user-generated content (like history or sessions) with an actionable path forward (e.g., a call-to-action button linking to another part of the application), rather than just stating the view is empty.
